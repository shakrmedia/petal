Quick Margins & Paddings

Use quick preset classes to add margins and paddings to any elements:

Margins
  • .m-xx : Margin (All directions)
  • .mv-xx : Vertical margins (top+bottom)
  • .mh-xx : Horizontal margins (left+right)
  • .mt-xx : Top margin
  • .mb-xx : Bottom margin
Paddings
  • .p-xx : Padding (All directions)
  • .pv-xx : Vertical paddings (top+bottom)
  • .ph-xx : Horizontal paddings (left+right)
  • .pt-xx : Top padding
  • .pb-xx : Bottom padding

Where xx is 5,10,15,20.
